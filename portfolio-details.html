Bob's Burgers Generator

Project information

  • Project date: May 2022
  • Project URL: https://github.com/mander-pander/bobsBurgers

This application allows users to create accounts and search through the available characters and burgers of the day from the show Bob's Burgers. Using HTML/CSS, JS, PostgreSQL and Express, the site is interactive and allows users to save favorites from each category.

RC-Social

Project information

  • Project date: July 2022
  • Project URL: https://github.com/mander-pander/rc-social

This application allows users to safely create accounts, socialize with other rc airplane enthusiasts, filter through airfields and save airplane models to a private wishlist for safekeeping.

Technologies used: React, Node.js, Express.js, Axios, PostgreSQL, Bootstrap, CSS Modules, bcrypt, JSON Web Token
